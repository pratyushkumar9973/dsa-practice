# dsa-practice
Daily Data Structures &amp; Algorithms practice for Data Engineering roles. Covers hash maps, arrays, strings, sliding window, two pointers,  stacks, queues, linked lists, trees, graphs, dynamic programming,  and SQL optimization. Each problem includes full explanation,  time/space complexity, and pattern recognition.

## 📚 Topics Roadmap

### Phase 1: Foundations (Week 1-4)
- [x] Hash Maps (counting, lookup, frequency)
- [ ] Arrays (two pointers, prefix sum)
- [ ] Strings (anagram, palindrome, pattern matching)
- [ ] Stacks & Queues (valid parentheses, monotonic stack)

### Phase 2: Intermediate (Week 5-7)
- [ ] Sliding Window (fixed, variable, two pointers)
- [ ] Linked Lists (reverse, merge, cycle detection)
- [ ] Binary Search (rotated array, search space)
- [ ] Sorting (merge, quick, heap, custom)

### Phase 3: Advanced (Week 8-10)
- [ ] Trees (BST, DFS, BFS, traversal)
- [ ] Graphs (BFS, DFS, topological sort, shortest path)
- [ ] Dynamic Programming (memoization, tabulation)
- [ ] Backtracking (permutations, combinations, subsets)

### Phase 4: SQL & System Design (Week 10-12)
- [ ] Advanced SQL (window functions, CTEs, recursive)
- [ ] Query Optimization (indexes, execution plans)
- [ ] Data Pipeline Design (batch vs streaming)
- [ ] Apache Airflow & Kafka basics
